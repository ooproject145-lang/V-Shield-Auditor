ABDULRAHMAN AHMED MUKHTAR
REG NO: MIU/22/CMP/CYB/090
Research Focus

The Challenge: Virtualization creates a complex security layer where simple misconfigurations lead to "VM Escape" and data exfiltration. Traditional security tools often fail to audit the Hypervisor layer itself.

The Solution: This framework implements an automated audit engine that parses VM Configuration Metadata. By identifying insecure side-channels (like Shared Clipboards and Remote Displays), we provide a hardened environment for secure operations.
